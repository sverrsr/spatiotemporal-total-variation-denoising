## Requirements
Matlab 2019a or newer. Older visions may be sufficient but have not been tested.

## Quick Start
- **Video denoising using simulated data.** Run [`sim_denoising.m`](https://github.com/THUHoloLab/spatiotemporal-total-variation-denoising/blob/master/main/sim_denoising.m) with default parameters.
- **Video denoising using simulated data.** First follow the instruction [here](https://github.com/THUHoloLab/spatiotemporal-total-variation-denoising/blob/master/data/experiment) to download the data. Then run [`exp_denoising.m`](https://github.com/THUHoloLab/spatiotemporal-total-variation-denoising/blob/master/main/exp_denoising.m) with default parameters.
